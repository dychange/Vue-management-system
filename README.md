# Vue-management-system   一个带用户验证的crud管理系统。
Vue-cli 3.0 / Express

## 简介

技术栈： 前端Vue-cli 3.0 + Element-ui ，后端Express + MySQL ， Axios做数据请求拦截 , jwt用户验证。

## Project setup
```
1. cd node-app
2. npm install
3. cd client
4. npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
